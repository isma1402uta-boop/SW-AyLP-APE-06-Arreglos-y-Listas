Repositorio correspondiente a la asignatura de **Algoritmos y Lógica de Programación (AyLP)** de la carrera de Ingeniería de Software. Contiene la resolución de los ejercicios prácticos (APE 06) enfocados en el manejo, comparación y optimización de estructuras de datos lineales estáticas y dinámicas.

## 📖 Descripción del Proyecto

Este proyecto tiene como propósito fundamentar las bases lógicas de programación en estudiantes de primer semestre. A través de la resolución de 10 ejercicios prácticos, se aborda la transición del uso de arreglos estáticos hacia colecciones dinámicas, utilizando bibliotecas estándar nativas:
* **Java:** `ArrayList` y `LinkedList`
* **C++:** `std::vector` y `std::list`

Los algoritmos desarrollados aquí sirven como base lógica para la futura elaboración de un aplicativo educativo orientado a la enseñanza interactiva del pensamiento computacional.

## ⚙️ Contenido y Ejercicios Desarrollados

El repositorio abarca los siguientes casos de estudio y desarrollo de software:
1. **Registros Básicos (ArrayList I):** Ingreso y validación de nombres de estudiantes y calificaciones.
2. **Algoritmos de Búsqueda:** Búsqueda lineal de productos en un inventario dinámico.
3. **Módulos CRUD (ArrayList II):** Creación, lectura, actualización y eliminación de registros mediante menús interactivos.
4. **Vectores Paralelos y Estructuras:** Sincronización de datos (Alumnos-Notas) e implementación de clases/structs para un sistema de inventario con cálculo de subtotales.
5. **Análisis de Rendimiento (Arreglos vs ArrayList):** Benchmarking de tiempos de ejecución (inserción masiva) e inmutabilidad vs. expansión de memoria.
6. **Estructuras Enlazadas (LinkedList):** Asignación de turnos y simulación de una fila de atención bajo el esquema **FIFO** (First-In, First-Out).

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Lenguaje Principal 1:** Java 17+ (Librería `java.util`)
* **Lenguaje Principal 2:** C++ 11+ (Librería STL: `<vector>`, `<list>`, `<chrono>`)
* **Lógica Computacional:** PSeInt (Pseudocódigo)
* **Control de Versiones:** Git & GitHub
